= 重設儲存 VM 的密碼
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
必要時重設儲存 VM 密碼。

.步驟
. 登入 Workload Factory 主控台。
. 在儲存設備中、選取 * 移至儲存設備詳細目錄 * 。
. 選取 * 儲存 VMS* 標籤。
. 在 Storage VMs （儲存 VM ）標籤中、選取儲存 VM 的三點功能表、以重設密碼。
. 選擇*重設密碼*。
. 在「重設密碼」對話方塊中、提供下列資訊：
+
.. * 新密碼 * ：輸入儲存 VM 的新密碼。
.. * 確認密碼 * ：再次輸入新密碼以確認。


. 按一下「 * 套用 * 」。
